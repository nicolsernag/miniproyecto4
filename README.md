# Miniproyecto4 - Battleship
## Description
This project is an implementation of the classic game Battleship, developed in Java using JavaFX for the graphical interface. The player places their fleet on a board and faces off against an automated enemy.

The project focuses on applying concepts of object-oriented programming, exception handling, data structures, and concurrency.

## Main Features
- Manual ship placement via drag and drop
- Automatic computer turns
- Game timer using threads
- Shooting system with visual feedback
- Victory and defeat screens

## Installation

To install the project, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/nicolsernag/borradorMiniproyecto3.git
2. Open the proyect in Intellij IDEA or your preferred IDE.
3. Build and run the application using the IDE or the command line:
./gradlew build
(or use the Run button in IntelliJ if it’s a JavaFX or console app.)
